init_BD.create_db(path=<class 'str'>, is_JV=<class 'bool'>)¶

This function create a database or open it if it already exists, and fill it with measurement information We put all the file’s information in a dictionary and then we write all the dictionary in the database, so we just call the db once> This is much faster

Parameters:

  • path (<str>) – path of the file

  • is_JV (<bool>) – True if the user wants to register J-V measurements, False otherwise

Return <list>:

a list containing all wafers that have been registered
